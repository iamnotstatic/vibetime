# vibetime

session analytics for the vibe coding era.
built for claude code, codex, and whatever comes next.

### install

```
npm install -g vibetime
```

### setup (once)

```
vibe init
```

### that's it

next time you close a claude session, vibe prints your summary.
no config. no account. your data lives in `~/.vibe/`.

### commands

```
vibe status              today's sessions
vibe log                 full history
vibe share               this week's card
vibe config show         current settings
```
